Mouse and Keyboard Controls:
	- Move: wasd / arrowKeys
	- Jump: space / up (Hold longer for more Height)
	- Sprint: RightShift?(why right actually?)
	- ZoomButton: F
	- RestartButton: R
	- ExitButton: ESC
	- ContinueButton: Enter on Endscreen of a mainLevel
	
Xbox Controls:
	- Move: Left JoyStick
	- Jump: A (Hold longer for more Height)
	- Sprint: B?
	- ZoomButton: Left/Right Trigger
	- RestartButton: Back Button(Select)
	- ExitButton: Start Button
	- ContinueButton: StartButton on Endscreen of a mainLevel
